== [0.1.0] - 2026-03-21

=== Phase 1 Complete

- Pattern detection for map/reduce/scan/scatter in source code
- Futhark codegen generating .fut files from detected patterns
- Build generator for multiple backends (opencl, cuda, multicore, c)
- GPU buffer management stubs via Zig FFI
- Bespoke manifest parsing for array operation annotations
- Integration tests for pattern detection and codegen

=== Added (scaffold — 2026-03-20)

- Initial project scaffold from rsr-template-repo
- CLI with subcommands (init, validate, generate, build, run, info)
- Manifest parser (`futharkiser.toml`)
- Codegen engine stubs
- ABI module (Idris2 proof type definitions)
- Library API for programmatic use
- Full RSR template (17 CI workflows, governance docs, bot directives)
- README.adoc with architecture overview
